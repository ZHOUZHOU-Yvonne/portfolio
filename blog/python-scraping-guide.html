Python爬虫实战：从零搭建电商数据采集系统

2026-06-10 · 12 min read

电商数据采集是很多企业的刚需——价格监控、竞品分析、市场调研都离不开数据。本文分享如何用Python搭建一套完整的电商数据采集系统，从单页面抓取到分布式爬虫，全部开源可复现。

1. 技术选型

  • requests + BeautifulSoup：静态页面抓取
  • Playwright：JavaScript渲染页面
  • Scrapy + Redis：分布式爬虫（大规模场景）
  • SQLite / PostgreSQL：数据存储
  • APScheduler：定时任务调度

2. 核心代码

2.1 基础爬虫框架

import requests
from bs4 import BeautifulSoup
import time, random

class BaseScraper:
    def __init__(self):
        self.session = requests.Session()
        self.session.headers.update({
            'User-Agent': 'Mozilla/5.0 ...'
        })

    def fetch(self, url, retries=3):
        for i in range(retries):
            try:
                time.sleep(random.uniform(1, 3))
                r = self.session.get(url, timeout=30)
                return BeautifulSoup(r.text, 'html.parser')
            except Exception as e:
                if i == retries - 1: raise
                time.sleep(2 ** i)

2.2 反爬虫策略

  1. User-Agent 轮换：维护20+个真实UA，每次请求随机选择
  2. 请求间隔：1-5秒随机延迟，模拟人类浏览行为
  3. IP代理池：集成免费/付费代理，自动剔除失效IP
  4. Cookie管理：自动维护登录态，定期刷新
  5. 验证码处理：接入打码平台API，自动识别滑块/图形验证码

3. 数据清洗与存储

原始HTML数据不能直接用，需要清洗和结构化：

import pandas as pd
import re

def clean_product_data(raw_items):
    products = []
    for item in raw_items:
        product = {
            'title': item.get('title', '').strip(),
            'price': float(re.sub(r'[^\d.]', '', item.get('price', '0'))),
            'sales': int(re.sub(r'\D', '', item.get('sales', '0')) or 0),
            'shop': item.get('shop', ''),
            'url': item.get('url', ''),
            'crawled_at': datetime.now().isoformat()
        }
        products.append(product)
    return pd.DataFrame(products)

4. 定时任务与监控

使用APScheduler配置定时采集，支持cron表达式。配合简单的日志和告警机制，确保爬虫稳定运行。

from apscheduler.schedulers.blocking import BlockingScheduler

scheduler = BlockingScheduler()

@scheduler.scheduled_job('cron', hour='*/6')

    scraper = EcommerceScraper()

5. 实战效果

以上方案在某电商项目中稳定运行6个月，日均采集50万+商品数据，准确率99.2%。核心经验：

  • 反爬策略要与目标网站匹配，不要过度设计
  • 数据清洗规则需要定期维护（网站会改版）
  • 监控比爬虫本身更重要——爬虫挂了你得知道
